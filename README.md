# BonkStars: Knockout Arena

A chaotic, fast-paced multiplayer game where players control goofy, wobbly characters trying to bonk each other off a shrinking platform using crazy physics and power-ups.

## Features

- **Simple to Learn:** Pick a character and start bonking in seconds.
- **Short Game Sessions:** Each match lasts 30 seconds – 3 minutes.
- **High Chaos:** Physics-driven gameplay with funny, surprising moments.
- **Unique Characters:** Each character has their own special Ultimate ability.
- **Dynamic Hazards:** Watch out for ice patches and trampolines!
- **Multiplayer Party:** Create a room, share a code, and play with friends.

## Getting Started

### Prerequisites

- Node.js & npm installed.

### Installation

1. Clone the repository:
   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. Install dependencies:
   ```sh
   npm i
   ```

3. Start the development server:
   ```sh
   npm run dev
   ```
